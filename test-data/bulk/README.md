# Bulk Onboarding Test Data

Place the following Excel template files in this directory before running bulk onboarding tests:

| File                  | Merchant Type     |
|-----------------------|-------------------|
| bulk_individual.xlsx  | Individual        |
| bulk_nida.xlsx        | NIDA Registered   |
| bulk_company.xlsx     | Company           |
| bulk_machinga.xlsx    | Machinga          |
| invalid_file.txt      | (negative test)   |

Download the correct templates from the MMP portal:
  Bulk Operations → Bulk Merchant Onboarding → Download Template
